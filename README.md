# C# Alqoritm Məşqləri

Bu qovluqda C# öyrənərkən yazdığım sadə alqoritm nümunələri yerləşir.

## Mövzular

- `for` dövrü
- `if` şərti
- `%` (mod) operatoru
- Cüt və tək ədədlərin yoxlanılması
- Ədədlərin cəminin hesablanması
- Müəyyən şərtə uyğun ədədlərin çap edilməsi

## Yazdığım tapşırıqlar

### 1. 1–49 arasındakı cüt ədədlərin cəmini hesablamaq
- Dövr vasitəsilə ədədlər üzərində gəzir.
- Cüt ədədləri seçir.
- Cəmi `total` dəyişənində saxlayır.

### 2. 1–49 arasında 15-ə tam bölünən ədədləri çap etmək
- `%` operatorundan istifadə edilir.
- Şərti ödəyən ədədlər ekrana yazdırılır.

### 3. FizzBuzz məşqi
- 3-ə bölünənlər → `Fizz`
- 5-ə bölünənlər → `Bazz`
- Hər ikisinə bölünənlər → `FizzBazz`

> Qeyd: Bu versiya məşq məqsədi ilə yazılıb. Gələcəkdə `if / else if / else` quruluşu ilə daha düzgün formada yenilənəcək.

### 4. 1–100 arasındakı tək ədədləri çap etmək
- `% 2 == 1` şərti ilə tək ədədlər seçilir.

### 5. 1–50 arasında 4-ə bölünən ədədləri çap etmək
- `% 4 == 0` şərti ilə uyğun ədədlər ekrana çıxarılır.

## Məqsəd

Bu layihənin məqsədi C# dilində əsas proqramlaşdırma məntiqini və alqoritmik düşünmə bacarığını inkişaf etdirməkdir. Məşqlər sadədən mürəkkəbə doğru davam etdiriləcək.
